# React Native Starter

Minimal Expo SDK 51 app with tabs navigation via expo-router, TypeScript strict mode, and dark/light theme support.

**Stack:** React Native | **Complexity:** simple
**Tags:** `mobile`, `expo`, `react-native`, `typescript`, `tabs`, `starter`

## Quick Start

```bash
npm install
npx expo start
```

## Claude Code
